## python+flask+mysql+vue
* install python
* install flask
* install mysql and set root account
* install vue and vue/cli
```shell script
$> sudo -i
$> export FLASK_APP=app.py
$> export FLASK_RUN_PORT=80
$> export FLASK_RUN_HOST=0.0.0.0
$> vue build index.vue && flask run
```
